Welcome to SM's documentation!
===============================

Simple Mind (SM)
--------------------------------------

Computer vision software platform that embeds convolutional neural networks (CNNs) inside a semantic network.

**SM** is a executing computer vision tasks/model 
based on machine learning and deep learning.

Getting Start!
^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^

.. code-block:: console
   
   from simplemind import sm 
   sm.runner(image_path, sm_model, output_dir)
